Plan → Visualise → Export

Sparrow Planner converts your workstream Excel plans into clean, editable PPT decks and a manager-ready dashboard. It's built to be safe (keeps your working generator), repeatable (local JSON retention), and simple (one folder, open via file://).
Generator (Stable)
Load Excel folder → select rows → validate → export editable PPT + updated JSON.
Dashboard (Manager view)
Read‑only charts, KPIs, hotspots. Uses same Excel folder + retention JSON.
Retention files live next to your Excel plans: selection-profile.json and sparrow-history.json.

Guided flow (MVP1)

The UI nudges users through the happy path: select folder → validate selections → export PPT → copy JSON back to folder.
Step 1
Select Excel folder
Loads .xlsx and looks for JSON beside them.
Step 2
Review & validate
Confirm selections per workstream, validate before export.
Step 3
Export PPT + JSON
PPT + updated JSON download via the browser.
Step 4
Copy JSON back
Copy JSON into the Excel folder to retain state.

MVP1 features

A stable baseline that's easy to extend without breaking what works.
Standalone

Runs from one folder via index.html. No install. No server.

PPT exports

Generate editable decks for detail plans, QA PoaP and exec summary.

Retention

Selections and colours persist via selection-profile.json.

History log

Export runs recorded in sparrow-history.json for trends.

MVP2 enhancements (planned)

Optional next steps once MVP1 is stable and adopted.
  • Weekly dashboard v2 with proper snapshots + week-on-week delta views
  • History-driven trend charts from sparrow-history.json (completion/QA/slips)

  • Board-pack export: auto-assembled PPT (exec summary + charts + hotspots)
