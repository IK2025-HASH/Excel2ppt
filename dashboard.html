Usage guidance

The dashboard is read‑only and never mutates data or profiles. It relies on whatever JSON and Excel plans exist in the selected folder.
  • Run the Generator first to refresh PPT and JSON.
  • Ensure you copy downloaded JSON back into the Excel folder.
  • Then open Dashboard and select the same folder for latest views.
